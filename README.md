<table>
<tr>
<td width="50%">

### ⚡ Approximate Computing
Hardware-software co-design for efficient inference. Designing approximate arithmetic units and mapping them to neural network layers — targeting embedded systems and edge AI.

</td>
<td width="50%">

### 🧠 Neural Network Compression
Structured & unstructured **pruning**, **quantization** (INT8/FP16), and **SVD-based decomposition** to reduce model size and inference cost on constrained hardware.

</td>
</tr>
<tr>
<td width="50%">

### 🌊 Reservoir Computing & RNNs
**Liquid State Machines (LSMs)** and sparse recurrent architectures. Exploring approximation strategies for reservoir connectivity to reduce compute while preserving dynamic expressiveness.

</td>
<td width="50%">

### 🤖 LLM Optimization & Agents
**RAG pipelines** (BM25, TF-IDF, dense retrieval), **fine-tuning** workflows, and **agent-based architectures**. Structured AI pipelines via **Knowledge Graphs** (SPARQL, Neo4j).
